Electron製の二画面ファイラー

### Requirement
- Electron 11.0.2
- React 17.0.1
- TypeScript 4.1.2
- Redux-Toolkit 1.5.0
- emotion 11.1.1

### Install Dependencies

```
$ cd knnfiler-nextron

# using yarn or npm
$ yarn (or `npm install`)

```

### Use it

```
# development mode
$ yarn dev (or `npm run dev` or `pnpm run dev`)

# production build
$ yarn build (or `npm run build` or `pnpm run build`)
```
